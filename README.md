# MATLAB to Python Model Conversion

This project demonstrates the conversion of a MATLAB-based numerical model into Python.

## Overview
The model simulates a damped oscillation system. Both MATLAB and Python implementations are provided.

## Features
- MATLAB to Python conversion
- Preserved computational behavior
- Output validation between implementations

## Files
- model.m
- model.py
- comparison.py

## Tools
- MATLAB
- Python (NumPy, Matplotlib)
